자동화 설정 방법 (플랫폼 선택)
1
위의 자동화 ON/OFF 토글을 켜세요.
2
이 탭을 열어 둔 상태로 유지하면 설정한 시각에 자동으로 Seibro API를 조회합니다.
3
브라우저 모드는 알림(Notification)으로 결과를 안내합니다. 이메일 발송은 Python 스크립트 방식을 이용하세요.
⚠️ Mac/Windows 스크립트 방식과 달리, 이 탭이 닫히면 자동화가 중단됩니다.
1
아래 파일을 같은 폴더에 준비: etf_etn_monitor.py, config.json
2
터미널에서 실행:
# 라이브러리 설치 (최초 1회)
pip3 install requests

# cron 등록 (매 평일 08:00)
(crontab -l 2>/dev/null; echo "0 8 * * 1-5 /usr/bin/python3 ~/etf_etn_monitor.py >> ~/etf_monitor.log 2>&1") | crontab -
3
등록 확인: crontab -l
1
📥 setup_windows.bat 다운로드 후 스크립트 폴더에 저장
2
관리자 권한으로 setup_windows.bat 실행 → Task Scheduler 자동 등록
3
작업 스케줄러 앱에서 "ETF_ETN_Monitor" 작업 확인
:: 수동으로 Task Scheduler 등록하는 명령어
schtasks /create /tn "ETF_ETN_Monitor" ^
 /tr "python %USERPROFILE%\etf_etn_monitor.py" ^
 /sc WEEKLY /d MON,TUE,WED,THU,FRI ^
 /st 08:00 /f
1
이 페이지를 홈 화면에 추가하세요 (Safari: 공유 → 홈 화면에 추가 / Chrome: 메뉴 → 홈 화면에 추가)
2
앱으로 열면 전체화면으로 사용 가능합니다.
3
위에서 자동화 ON → 시각이 되면 브라우저 알림이 옵니다. (탭이 열려 있어야 함)
4
이메일 발송은 PC에서 Python 스크립트를 cron/Task Scheduler로 돌리는 것을 권장합니다.
✅ 이 어드민 페이지 자체는 Mac·Windows·모바일 모든 환경에서 사용 가능합니다.
